# README

An app to manage schema definitions within Contentful.

## Libs

The following 3rd party libraries are used:

### contentful-management

Contentful library for content management.

[Download Here](https://github.com/contentful/contentful-management.js)

-- OR --

To install, execute the following npm command:

    npm install contentful-management --save

### marked

Recommended library for parsing markdown.  The library is the same used by Contentful.

[Download Here](https://github.com/chjj/marked)

-- OR --

    npm install marked --save
