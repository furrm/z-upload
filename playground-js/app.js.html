Best Practice

This document contains inline JavaScript capturing the following best practices:

  • Don't use the <with statement
  • Always use semicolon to end statements
  • Left versus right curly braces
  • Always use === never use ==
  • Don't use multiline string literals.
  • Avoid global variables. Use THIS_CONVENTION for naming global variables when required.
  • Classes should use ThisConvention when being defined.
  • Avoid using x++ to increment. Instead, use x += 1
  • Always put curly braces around if statements.
